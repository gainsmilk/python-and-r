# Python 13 Lists and Loops

Category: `python`

## Assignment 3: Lists and Loops

## Tasks

1. Create a list of five fruits. Print the second and last fruit.
2. Add a fruit to the list, then remove the first fruit.
3. Use a for loop to print each item in the list.
4. Create a while loop that prints numbers from 1 to 10.

## Tips

- Lists use zero-based indexing (`fruits[0]` is the first item).
- `.append(item)` adds an item; `.remove(item)` removes an item.
- A while loop runs as long as a condition is `True`.
