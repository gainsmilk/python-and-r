Category: `python`

## Assignment 1: Python Syntax and Variables

## Tasks

1. Create a Python script that prints `"Hello, Python!"`.
2. Declare three different variables:
- one integer
- one float
- one string
   Assign them values and print their types using `type()`.
3. Swap the values of two variables without using a temporary variable.
4. Write a short comment explaining what the `#` symbol does in Python.

## Tips

- Use `print()` to display output.
- Use `type(variable)` to check data types.
- Python allows swapping two variables in one line using `a, b = b, a`.
- Comments in Python start with `#` and are ignored by the interpreter.
